# Pusher Real Time Chat App

A messenger clone where users can create groups and individual conversations and everything is real-time with pusher.

https://real-time-chat-app-rajvira10.vercel.app/

## Technologies:

1. NextJS 13
2. TypeScript
3. MongoDB
4. Prisma
5. Tailwind
6. NextAuth
7. Pusher
8. Zustand
